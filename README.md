# Letermovir
This repository contains data and scripts supporting the manuscript "Discovery of a Low-Cytotoxicity Letermovir Analogue with Enhanced Binding Affinity to the Human Cytomegalovirus Terminase Subunit pUL56: In silico and In vitro Study".

# Contents 
'MTT+substance_database' - compound database and MTT assay results
'Molecular_docking/' - input files, poses, docking results
'Molecular_dynamics/' - input files for MD simulations (parameters, topology, scripts)
